# AI Gym & Fitness Assistant

## Project Overview

AI Gym & Fitness Assistant is an AI-powered fitness application developed using Streamlit, Python, SQLite, and Groq AI. The application helps users track workouts, monitor BMI, analyze nutrition, and receive AI-powered fitness guidance.

## Features

- AI Fitness Chatbot
- User Authentication (Login & Signup)
- Nutrition Intelligence & BMI Analysis
- Workout Logger
- Workout History Tracking
- Fitness Habit Tracker
- AI Gym Planner
- Performance Score Analyzer
- Fitness Analytics Dashboard
- BMI History Tracking

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- Groq AI

## Project Structure

- app.py
- fitness.db
- requirements.txt
- README.md

## Installation

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py
